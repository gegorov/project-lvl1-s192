## Setup

```
make install
```
for global install:
```
sudo npm install -g brain-games-ge
```

### Run

to start Brain Games:
```
make start
```
to start Brain Even Game:
```
make start-even
```
to start Brain Calc Game:
```
make start-calc
```
to start Brain GCD Game:
```
make start-gcd
```
to start Brain Balance Game:
```
make start-balance
```
#### Run with global installation
to start Brain Games:
```
brain-games
```
to start Brain Even Game:
```
brain-even
```
to start Brain Calc Game:
```
brain-calc
```
to start Brain GCD Game:
```
brain-gcd
```
to start Brain Balance Game:
```
brain-balance
```
